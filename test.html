GET /health — 健康检查。Docker 健康检查使用此端点。无需认证。
POST /v1/chat/completions — OpenAI 兼容的 Chat 对话接口。支持多轮对话、function calling、thinking 模式等。
模型列表: agnes-1.5-flash agnes-2.0-flash
POST /v1/chat/completions (stream: true) — 流式 Chat 对话。返回 SSE 事件流，实时输出文本。
POST /v1/messages — Anthropic Messages API 兼容接口。网关会将 Anthropic 格式转换为 OpenAI 格式发送给 Provider。
认证方式: x-api-key header 优先，或 Authorization: Bearer。
POST /v1/messages (stream: true) — Anthropic 流式 Messages。返回 Anthropic 格式的 SSE 事件流（message_start, content_block_delta, message_stop 等）。
POST /v1/images/generations — 文生图接口。支持 url 和 b64_json 两种响应格式。
模型: agnes-image-2.0-flash agnes-image-2.1-flash
POST /v1/images/edits — 图生图 / 图片编辑接口。需要提供原图和编辑提示词。
图片可以是 URL、Data URI 或 base64 字符串。网关会将请求转换为 Provider 的 /images/generations 格式。
POST /v1/videos/generations — 异步视频生成接口。支持文生视频、图生视频。
返回 video_id，需轮询 GET /v1/videos/:id 获取结果。
模型: agnes-video-v2.0
GET /v1/videos/:id — 查询视频生成状态。状态值: queued → processing → completed / failed。
可点击"自动轮询"按钮，每 3 秒查询一次直到完成。
